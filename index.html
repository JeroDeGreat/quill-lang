      Get Started
      
Install Quill in 60 seconds

      
One command on any platform. No configuration needed.

    

    

      
      
      
      
    


    

      

        

          
1

          

            
Download the zip

            Download quill-lang-2.0.0.zip
            
Extract to your Downloads folder.

          

        

        

          
2

          

            
Open PowerShell

            
Press Win+X → Terminal (or Windows PowerShell)

          

        

        

          
3

          

            
Navigate and run

            

              cd "C:\Users\YourName\Downloads\quill-lang"

              powershell -ExecutionPolicy Bypass -File ".\install.ps1"
            

          

        

        

          
4

          

            
Open a new terminal and verify

            
quill version

          

        

      

      

        
⚠️ Common Issues

        
    
          
  • Script not digitally signed: Use powershell -ExecutionPolicy Bypass -File ".\install.ps1"
    • 
          
  • quill not found: Close terminal and open a NEW one — the PATH needs to reload
    • 
          
  • Python not found: Install Python from python.org — tick "Add Python to PATH" on install
    • 
          
  • Space in username errors: Always wrap paths in double quotes in PowerShell
    • 
        

      

    


    

      

        
Linux / Kali Linux — One line install

        curl -sSL https://quill-lang.org/install.sh | bash
        


        
Kali Linux optimized

        sudo apt update && sudo apt install python3 python3-pip python3-tk -y

        pip3 install quill-lang customtkinter matplotlib
        


        
From zip

        cd ~/Downloads/quill-lang

        chmod +x install.sh && ./install.sh

        source ~/.bashrc

        quill version
      

      

        
⚠️ Common Issues on Linux

        
    
          
  • command not found: Run source ~/.bashrc
    • 
          
  • No module tkinter: sudo apt install python3-tk
    • 
          
  • GUI won't start on Kali: sudo apt install python3-dev python3-tk
    • 
        

      

    


    

      

        curl -sSL https://quill-lang.org/install.sh | bash

        source ~/.zshrc

        quill version
      

    


    

      

        pip install quill-lang

        pip install customtkinter matplotlib  # Optional: modern GUI + charts
      

    

  







  

    

      Language Tour
      
Everything you need, in plain English

    

    

      

        
Variables

        
let name be "Alice".
let age be 25.
let active be yes.
set age to 26.

      

      

        
Arithmetic

        
let total be price times qty.
let half be total divided by 2.
let rem be total mod 7.
let root be square root of 144.

      

      

        
Conditions

        
if age is at least 18.
  say "Adult.".
else if age is 17.
  say "Almost!".
else.
  say "Minor.".
end if.

      

      

        
Loops

        
repeat 5 times.
  say "Hello!".
end repeat.

for each item in items.
  say item.
end for.

      

      

        
Functions

        
define greet taking name.
  say "Hello, {name}!".
end define.

greet with "Alice".

      

      

        
Lists & Maps

        
let fruits be a list of
  "apple", "banana".
let p be a map with
  "name" as "Alice".

      

      

        
HTTP Requests

        
let data be get
  "https://api.example.com".
let json be parse data as json.
say json at "field".

      

      

        
Database

        
connect to database "app.db" as db.
execute "INSERT ..." on db.
let rows be query db with "SELECT...".

      Examples
      
See Quill in action

    

    

      
      
      
      
      
    

    

      

        

          

            
hello.ql

            
say "Hello, World!".

let name be "Alice".
let age be 25.

if age is at least 18.
  say "Welcome, {name}!".
else.
  say "Sorry, too young.".
end if.

          

        

        

          
Output

          
Hello, World!
Welcome, Alice!

        

      


      

        

          

            
counter.ql

            
use gui.
set theme to "dark".
open a window titled "Counter"
  with width 300, height 200.

let count be 0.
create a label named countLabel
  saying "Count: 0".
create a button named addBtn
  saying "Add 1"
  that calls addOne.

define addOne.
  add 1 to count.
  set the text of countLabel
    to "Count: {count}".
end define.
show window.

          

        

        

          
Result

          

            
Counter

            

              
Count: 0

              
            

          

        

      


      

        

          

            
classifier.ql

            
use ml.

let emails be a list of
  "free money win prize click",
  "meeting at 9am tomorrow".

let labels be a list of
  "spam", "ham".

let model be
  train naive bayes emails, labels.

let result be
  predict naive bayes model,
  "claim your free gift now".

say "Classification: {result}".

          

        

        

          
Output

          
Classification: spam

        

      


      

        

          

            
todos.ql

            
connect to database "todos.db"
  as db.

execute
  "CREATE TABLE IF NOT EXISTS
  todos (id INTEGER PRIMARY KEY,
  task TEXT)"
  on db.

execute
  "INSERT INTO todos (task) VALUES (?)"
  on db with a list of "Buy milk".

let rows be
  query db with "SELECT task FROM todos".

for each row in rows.
  say row at "task".
end for.

          

        

        

          
Output

          
Buy milk

        

      


      

        

          

            
github.ql

            
let url be
  "https://api.github.com"
  joined with "/repos/python/cpython".

let response be get url.
let data be
  parse response as json.

say "Repo: {data at \"full_name\"}".
say "Stars: {data at \"stargazers_count\"}".
say "Language: {data at \"language\"}".

          

        

        

          
Output

          
Repo: python/cpython
Stars: 62,000+
Language: Python
